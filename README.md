# vending_machine


Getting started

```
bundle install
ruby app.rb
```


Testing

```
rake calculate_change_service    # Launch tests for calculate_change_service
rake coin                        # Launch tests for coin
rake controllers                 # launch all controller tests
rake models                      # Launch all model tests
rake product                     # Launch tests for product
rake repositories                # launch all repository tests
rake services                    # launch all service tests
rake spec                        # Run RSpec code examples
rake vending_machine_controller  # Launch tests for vending_machine_controller
```
